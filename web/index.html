Cairns

Your notes, straight to your GitHub repo.

Capture on Mac and iOS. Plain markdown commits. No server, no subscription.

Download for Mac

What it is

Cairns is the capture layer: a blank page that's always one keystroke away, with everything landing as markdown commits in a GitHub repository you own. Heavy editing happens wherever you already work — your IDE, Obsidian, an agent running against the clone.

Features

  • Quick capture ⌘⇧N

    A capture window from anywhere on Mac; open the iOS app to a ready-to-type blank note. Type or dictate.

  • Your repo, your notes

    Every note is a markdown file committed to a GitHub repo you own. Full git history, nothing proprietary.

  • Works offline

    On Mac, notes are local git commits that push when they can; on iOS they queue durably and drain automatically.

  • Native state

    Your GitHub token lives in the Keychain. No browser storage, nothing for the OS to evict.

  • Agent-ready

    Plain markdown in git. Claude Code, Codex, and your own scripts can read and operate on every note.

  • Auditable

    No telemetry, no tracking, no subscription — GitHub already syncs your files for free. Read every line.

Install

Pre-signing distribution while Cairns finds its audience — App Store / TestFlight come later.

Mac

Download the .dmg from the latest release.

First launch: right-click the app and choose Open to get past Gatekeeper (unidentified developer).

iOS

Sideload the .ipa from the release with AltStore or Sideloadly.

Or clone the repo and run the ios/ app on your device from Xcode with a free Apple ID.
